Focus Task App
Custom timer

Use the Pomodoro technique to push through each task or set a custom time.

Gifs from Giphy API

Each time a task is completed a success Gif is diplayed. Tasks that are uncompleted are shown failed gifs.

Increase focus

Focusing on one task at a time is a proven way to get things done. Multi-tasking and context switching can be detrimental to your productivity.

One Task

Focus Task App is an app made for focusing on one task at a time. A custom timer allows you to use the Pomodoro Method or custom time sprints. After every task a random gif is displayed from the Giphy API.
